Cinematics

  • A Spring Boot Application that gives basic information about movies, based on user's queries.
  • Tech Stack involved: Spring Framework, Hibernate ORM, Java, PostgreSQL DB as backend, and Thymeleaf Engine, HTML, CSS and JavaScript as frontend.
  • Users can search for their favourite movies, actors and directors and view corresponding details.
  • Application has custom movie search abilities based on language, genre and can filter and sort the results based on any movie criteria.

SocketChat

  • An online chat application based on SocketIO and Express Javascript Frameworks.
  • Provides environment for real-time communication for multiple users at the same time.
  • Can be deployed online using Localtunnel, to enable chatting and usage of application from many endpoints.
  • Made using HTML for markup, Bootstrap CSS for styling and Javascript for functionality.

Text-to-Speech Synthesis

  • A Javascript web application that uses Web Speech API to convert typed text into speech.
  • Provides literal word-to-word speech conversion that can be altered to any specific rate and pitch.
  • Users can also choose among different voices for text-to-speech output.
  • Made using HTML for markup, Bootstrap CSS for styling, Popper JS and JQuery for functionality.
