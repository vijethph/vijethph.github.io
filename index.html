FeedHub

  • This is a News Aggregator and RSS Reader app built with Vue.js and Firebase which uses Firebase Auth and Azure AD to implement Google Sign-in, Microsoft Sign-in and password reset features.
  • Adopted Firestore Database to save and update user's favourite news sources and RSS feeds, and implemented an Azure Function to receive RSS site URLs, fetch latest RSS feeds, parse them and send feed data to the user.
  • Leveraged GitHub Actions to create a fully automated CI/CD pipeline to deploy the app to Firebase Hosting, OpenWeatherMap API to get current weather and NewsAPI to fetch latest news.
  • Tech Stack employed: Vue.js Framework, Firebase Auth, Firestore DB, Azure Functions.

PlacementCell

  • This is a MERN Stack App that provides a recruitment training platform for students, with attributes like user authentication, discussion forum, quiz, videos playlist and dashboard for company updates.
  • Formulated a quiz component with multiple choice questions, timer alerts, result analysis and notification features, and a discussion forum that allows students to edit and delete posts and comments, and a video player with aptitude practice videos playlist
  • Leveraged GitHub Actions to create a fully automated CI/CD pipeline to deploy the app to Azure App Service, and HackerEarth API to create a functional code compiler with support for C, C++, Java and Python Languages.
  • Tech Stack involved: MongoDB, Express and React Frameworks, Node.js runtime.

Cinematics

Text-to-Speech Synthesis

  • A Javascript web application that uses Web Speech API to convert typed text into speech.
  • Provides literal word-to-word speech conversion that can be altered to any specific rate and pitch.
  • Users can also choose among different voices for text-to-speech output.
  • Made using HTML for markup, Bootstrap CSS for styling, Popper JS and JQuery for functionality.

Credit Card Validation

  • A JavaFX program that uses an implementation of Luhn's Algorithm to determine the validity of credit card numbers.
  • It captures the image of credit card using Webcam Capture API, parses the digits in the image using Tesseract OCR, and gives the corresponding credit card number as input to the validation procedure.
  • User is allowed to change/edit the parsed digits in the credit card number before performing validity check.
  • Tech Stack involved: JavaFX, Webcam Capture API, JavaCPP Presets library, Tesseract OCR.

Barcode Scanner

  • A Kivy Application, and its equivalent Android app that detects QR codes and barcodes in real-time, by accessing device camera.
  • Parses all forms of QR codes and barcodes into text and allows the user to search for the text in their respective Web Browser.
  • Has additional functionalities to pause, restart and stop the device camera.
  • Made with Kivy Framework, OpenCV and ZBar libraries, and implemented in Python and Android.
